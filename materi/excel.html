Excel

Microsoft Excel adalah software yang digunakan untuk pengolahan angka

Tentang Materi

Microsoft Excel merupakan perangkat lunak pengolah angka berupa spreadsheet. Excel digunakan untuk bekerja dengan daftar data, menghitung angka-angka, membuat laporan, diagram, grafik, dan sebagainya.

Materi yang telah dipalajari

  • Fungsi Aritmatika
  • Fungsi Count
  • Fungsi IF Tunggal & IF Bertingkat
  • Fungsi IF AND & IF OR
  • Fungsi Left, Mid, dan Right

1. Fungsi Aritmatika

Aritmatika adalah bagian dasar matematika yang mempelajari operasi bilangan seperti Penambahan, Pengurangan, Perkalian, dan Pembagian.

Lambang Aritmatika yang biasa digunakan pada Ms.Excel adalah :

Simbol Keterangan
+ Penambahan
- Pengurangan
* Perkalian
/ Pembagian
^ Kuadrat
  • SUM digunakan untuk menjumlahkan nilai pada sel

    Rumus : =SUM(Sel Awal : Sel Akhir)

  • AVERAGE digunakan untuk menghitung nilai rata-rata pada sel

    Rumus : =AVERAGE(Sel Awal : Sel Akhir)

  • MIN digunakan untuk menghitung nilai Terkecil pada sel

    Rumus : =MIN(Sel Awal : Sel Akhir)

  • MAX digunakan untuk menghitung nilai Terbesar pada sel

    Rumus : =MAX(Sel Awal : Sel Akhir)

  • RANK digunakan untuk mencari peringkat melalui perbandingan nilai pada masing-masing sel

    Rumus : =RANK(Sel Awal : Sel Akhir)

    Catatan : Klik Fn + F4 untuk mengunci nilai

2. Fungsi Count

  • COUNT digunakan untuk menghitung jumlah sel yang terisi data

    Rumus : =COUNT(Sel Awal : Sel Akhir)

  • COUNTBLANK digunakan untuk menghitung jumlah sel yang tidak terisi data

    Rumus : =COUNTBLANK(Sel Awal : Sel Akhir)

3. Fungsi IF Tunggal dan IF Bertingkat

Adalah fungsi yang akan menghasilkan suatu nilai jika suatu kondisi terpenuhi dan akan menghasilkan nilai lainnya Ketika suatu kondisi tersebut tidak terpenuhi

  • Rumus IF Tunggal : =IF(TesLogika;KondisiBenar;KondisiSalah)
  • Rumus IF Bertingkat : =IF(TesLogika1;KondisiBenar1;IF(TesLogika2;KondisiBenar2;KondisiSalah))

3. Fungsi IF AND dan IF OR

IF AND kedua uji haruslah bernilai benar, maka kondisi akan terpenuhi
IF OR salah satu uji haruslah bernilai benar, maka kondisi akan terpenuhi

  • Rumus IF AND : =IF(AND(TesLogika1;TesLogika2;...);KondisiTerpenuhi;KondisiTidakTerpenuhi)
  • Rumus IF OR : =IF(OR(TesLogika1;TesLogika2;...);KondisiTerpenuhi;KondisiTidakTerpenuhi)

4. Fungsi Left, Mid, Right

Left digunakan untuk mengambil dan menampilkan isi sel berupa karakter/kata yang dihitung mulai dari arah kiri

  • Rumus : =LEFT(Sel Asal; Jumlah karakter yang akan ditampilkan)

MID digunakan untuk mengambil dan menampilkan isi sel berupa karakter/kata yang dihitung mulai dari indeks karakter pilihan, lalu berapa karakter yang akan ditampilkan mulai dari indeks karakter pilihan tersebut.
  • Rumus : =MID(Sel Asal; Dimulai dari; Jumlah karakter yang ditampilkan)
Right digunakan untuk mengambil dan menampilkan isi sel berupa karakter/kata yang dihitung mulai dari arah kanan
  • Rumus : =RIGHT(Sel Asal; Jumlah karakter yang ditampilkan)

5. VLOOKUP dan HLOOKUP

VLOOKUP digunakan untuk pengisian data berdasarkan tabel referensi yang berbentuk kolom atau vertikal.

  • Rumus : =VLOOKUP(Nilai Kunci; Tabel Referensi; Kolom yang digunakan; 0)

HLOOKUP digunakan untuk pengisian data berdasarkan tabel referensi yang berbentuk kolom atau vertikal.
  • Rumus : =HLOOKUP(Nilai Kunci; Tabel Referensi; Kolom yang digunakan; 0)

6. Diagram

  • Buatlah tabel dan isi terlebih dahulu
  • Blok tabel yang akan digunakan untuk menjadi diagram
  • Pilih menu Insert, lalu cari Chart
  • Pilih Column Chart sesuaikan dengan kebutuhan
  • Diagram akan muncul secara otomatis

7. Pivot Table

Adalah sebuah tabel interaktif yang dapat menampilkan ringkasan data berjumlah besar dalam bentuk dan orientasi yang berbeda.
Pivot Table juga mampu melakukan kalkulasi pada setiap item yang dibutuhkan dengan menggunakan cara perhitungan sesuai kebutuhan.

  • Buatlah table yang berisikan data, usahakan tidak boleh ada data yang kosong, lalu setelah itu blok table tersebut
  • Pilih menu Insert, setelahnya klik PivotTable, maka akan muncul menu seperti ini:
  • Di menu PivotTable, pilih Existing Worksheet lalu pada Location, klik tombol panah untuk menunjuk lokasi munculnya PivotTable.
  • Klik Sel kosong agar PivotTable muncul, Setelahnya klik Kembali Tombol panah. Lalu klik OK
